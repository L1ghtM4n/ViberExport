# :calling: ViberExport
Export messages from Viber messenger using viber.db file

### :zap: Usage:
`python3 export.py viber.db`

### :comet: Info:
viber.db is located at
`C:/Users/<User>/AppData/Roaming/ViberPC/<Phone>/viber.db`
